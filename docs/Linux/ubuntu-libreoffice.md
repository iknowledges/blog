# Ubuntu安装LibreOffice

1. 安装libreoffice

下载deb安装包：https://www.libreoffice.org/download/download-libreoffice/

```
cd /path/to/DEBS/
sudo dpkg -i *.deb
```

2. 卸载libreoffice

```
sudo apt-get purge libreoffice? libobasis?
```
