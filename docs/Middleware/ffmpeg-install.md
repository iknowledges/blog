# FFmpeg安装教程

1. 下载[ffmpeg-release-essentials.zip](https://www.gyan.dev/ffmpeg/builds/)，并解压到指定目录。

2. 将/path/to/FFmpeg/ffmpeg-7.1.1-essentials_build/bin加入到环境变量Path。

3. 在命令行输入下面命令测试是否安装成功：

```
ffmpeg -version
```
