# 安装Python-LZO

## Ubuntu

```
sudo apt install liblzo2-dev
pip install python-lzo
```

## Windows

1. 安装Visual Studio，【工作负荷】勾选【使用C++的桌面开发】，并确保勾选了【适用于x64/x86的MSVC生成工具】和【Windows 11 SDK】

2. 下载[python-lzo-1.16.zip](https://github.com/jd-boyd/python-lzo/tags)，并使用下面命令安装：

```
pip install python-lzo-1.16.zip
```
